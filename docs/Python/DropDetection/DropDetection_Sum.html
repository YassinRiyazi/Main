DropDetection_Sum Documentation

walk_forward()
Walk forward from the start index until we find a value greater than 0. Steep: a parameter that defines the steepness of the slope to detect the drop.
backward()
Walk forward from the start index until we find a value greater than 0. Steep: a parameter that defines the steepness of the slope to detect the drop. if want to include wider dropes with low slope decrese the sensitivity of the slope. For example, if you want to detect drops with low surface tention, surfactant, slope shold be less than 0.0025
detect_drop()
Trying to detect images with less than 2ms delay. [V] 0. Convert image to grayscale, [V] 1. Resizing image, [V] 2. Applyng gussina blur (morphologyEx worked better than Gaussian blur), [-] 3. Transpoing image (Optimization puposes) [Didn't improve any thing and because of if damaged was more than benefits], [V] 4. Summation over rows [-] 5. Normalize images based on height and maximum brightness, [V] 6. Finding beggining of the drop and ending of the drop, and finally drawing a rectangle around the drop.
caution:
Code will fail if there are more than one drop in the image.
Images should have exactly 5 rows of black pixels at the bottom of the image.
It works with tilted setup drop imaeges, on other drop shape it is untested
draw_bounds()
Draw a rectangle on the image from start to end.
