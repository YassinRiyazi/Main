PyParser

parse_python_docstring(docstring:str,)
Parse a Google-style Python docstring into a structured dictionary. Recognized sections include: Args, Returns, Raises, Examples, Notes, See Also, Caution, and Warning.
Parameters:
  • docstring (str): A raw docstring from a Python function.
Returns:
'warning': list of str,
examples:
>>> doc = '''
... Adds two numbers.
... Args:
... a (int): First number.
... b (int): Second number.
... Returns:
... int: Sum of the numbers.
... '''
>>> parse_python_docstring(doc)
extract_python_functions(file_path:str,)
Extract top-level Python functions and their docstrings from a source file. Uses the AST (Abstract Syntax Tree) module to safely parse function names and docstrings.
Parameters:
  • file_path (str): Path to the Python source file (.py).
Returns:
'doc': dict # Parsed result from parse_python_docstring()
examples:
>>> extract_python_functions("my_script.py")
[{'name': 'foo', 'doc': {...}}, ...]
